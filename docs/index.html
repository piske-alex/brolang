Compiler Pipeline

.bro source → Lexer → Tokens → Parser → AST → Compiler → Bytecode → Stack VM

No transpilation. No eval(). Source is lexed, parsed into an AST, compiled to real bytecodes (CONST, ADD, MUL, JMP_FALSE, CALL, RETURN), and executed on a stack-based virtual machine. Same architecture as CPython. More vibes.
