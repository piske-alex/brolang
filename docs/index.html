# BroLang — llms.txt

> Compiled language with crypto-bro keywords, bytecode VM, native x86-64 backend.

## Syntax
gm/gn: program start/end. ser x is 5: variable. hodl PI is 3.14: constant.
based name(params) ... gg: function. paper_hands value: return.
wagmi condition ... ngmi ... fr: if/else/endif.
to_the_moon condition ... gg: while loop.
sweep item in arr ape_in ... ape_out: for-each.
shill "text" or shill "hello {name}": print (with interpolation).
anon(x) x times 2: lambda. value pump fn: pipe (|>).
ape_into degen_math: import module.
ape_in/ape_out: { } block delimiters. STOP RUG.: alt program end.

## Operators (text-based)
plus minus times divided_by mod: arithmetic
equals greater_than less_than gte lte: comparison
and or not: logic. is: assignment. few/cope: true/false.

## Modules (via ape_into)
degen_math: floor ceil sqrt abs sin cos pow min max random pi
degen_string: char_at str_len upper lower trim substr replace split join contains
degen_array: arr_new arr_len arr_get arr_set arr_push arr_pop arr_sort
degen_time: now sleep

## Example
```
gm
ape_into degen_math
hodl FEE is 0.003
ser balance is 1000
based apply_fee(amount, rate)
  paper_hands amount times (1 minus rate)
gg
shill "After fee: {balance pump apply_fee(FEE)}"
gn
```
